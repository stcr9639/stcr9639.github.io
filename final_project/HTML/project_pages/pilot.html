Abstract

Real world pilot training can be an expensive and dangerous endeavor. Through advances in virtual reality technology, it has become possible to pair the physical effects of real world roller coasters with imaginary virtual worlds. In doing so, we can create a realistic simulated experience at a low cost. We have been tasked with completing a preliminary design for a roller coaster flight simulator to be used for pilot training. To do this, we used Matlab to create re-configurable coaster sections that would be analogous to specific flight situations. Using dynamics equations, we also were able to derive equations for the velocities and accelerations throughout the track elements. This gives us the ability to precisely calculate the g's that a pilot in the simulator would experience in flight, and create a virtual environment to match that experience.
