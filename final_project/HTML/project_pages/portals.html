Final Deliverable

The core interaction implemented was teleportation. Teleportation is described as an element that allows a player to instantly travel between two locations or levels. In my project teleportation is implemented in the following four different ways: ray-based teleportation, object-based teleportation, scene changing, and portals. Ray-based teleportation allows a user to point a ray in the direction they wish to go and it moves the player a few feet forward. This effect was achieved by using the ray-interactors that are a part of the XR Rig extension. The object-based teleportation allowed a user to touch an object and teleport to a different location on the map. The scene change allowed for a player to “enter the houses” which was a different scene than the open terrain scene they started in. The portal allowed for the player to see through the portal using a camera linked to the player's movements and would allow the player to walk through to another location in the scene.
